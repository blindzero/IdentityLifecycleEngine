# GitHub Pages setup

This repository can publish the documentation as a website using **GitHub Pages**.

## Recommended configuration

1. In GitHub, go to **Settings**.
2. Open **Pages**.
3. Under **Build and deployment**, select:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/docs`
4. Save.

GitHub Pages will publish `docs/index.md` as the site entry point.

## Notes

- The documentation is written in Markdown.
- The website output is generated by GitHub Pages.
- Keep Markdown pages short and linkable.
